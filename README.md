sinatra-websocketio
===================

* Node.js like WebSocket I/O plugin for Sinatra.
* http://github.com/shokai/sinatra-websocketio


Installation
------------

    % gem install sinatra-websocketio


Requirements
------------
* Ruby 1.8.7+ or 1.9.2+
* Sinatra 1.3.0+
* EventMachine
* jQuery
